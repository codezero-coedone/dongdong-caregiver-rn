## 화면 구성

### 헤더
- 뒤로가기 버튼 (`<`)
- 타이틀: "프로필 수정"

### 1. 회원정보 섹션
- 이름 + 본인인증 배지
- 생년월일 (읽기 전용)
- 휴대폰 번호 (수정 가능, 본인인증으로 변경)
- 주소 (주소 검색 + 상세주소)
- 범죄경력회보서 (파일 업로드, 선택)

### 2. 경력 섹션
- 신입 / 경력 토글 버튼

### 3. 자격증 섹션
- 요양보호사
- 간호조무사
- 산후관리사
- 간병사(민간자격증)
- 기타 자격증

### 4. 하단 버튼
- "수정하기" 버튼 (전체 화면 너비)

---

## 기술 스택

- **React Native** + Expo Router
- **react-hook-form** + zod (폼 검증)
- **NativeWind** (TailwindCSS 스타일링)
- **StyleSheet** (복잡한 스타일)

---

## 파일 구조

```
app/
├── profile/
│   ├── _layout.tsx       # Stack Navigator
│   ├── edit.tsx          # 프로필 수정 스크린
│   └── (주소 검색은 1차 범위 제외 / WebView 사용 금지)
└── (tabs)/
    └── my.tsx            # 프로필 수정 버튼 연결
```

## 관련 컴포넌트 재사용

| 컴포넌트 | 용도 |
|---------|------|
| `Input` | 휴대폰, 상세주소 입력 |
| `AddressInput` | 주소 검색 UI |
| `FileUploadBox` | 범죄경력회보서 업로드 |
| `Button` | 수정하기 버튼 |
| `Header` | 화면 헤더 (필요시) |
