## 화면 구성

### 헤더
- 뒤로가기 버튼 (`<`)
- 타이틀: "자기 소개"

### 1. 자기소개 텍스트 입력
- TextInput (multiline)
- Placeholder: "나를 표현할 소개를 작성해주세요."
- 글자 수: 0/200
- 오른쪽 "입력" 버튼

### 2. 나만의 강점 (0/3)
선택 가능한 태그 (최대 3개):
- 친절함, 책임감, 위생적인, 안전제일
- 적극적, 인내심, 자상함, 믿음직, 세심함

### 3. 자신 있는 돌봄 (0/5)
선택 가능한 태그 (최대 5개):
- 골절, 재활치료, 치매, 섬망
- 파킨슨, 욕창, 외상 환자, 석션
- 피딩, 소변줄, 장루, 기저귀 케어
- 투석, 좌우 편마비, 하반신 마비
- 전신 마비, 전염성 질환, 정신 질환
- 자가 보행, 부축

### 4. 하단 버튼
- "등록하기" (선택 완료 시 활성화)
