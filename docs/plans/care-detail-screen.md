## 화면 구성

### 헤더
- 뒤로가기 버튼 (`<`)
- 타이틀: "진행 중인 간병 상세보기"

### 1. 환자 정보 섹션
- 남은 일수 배지 (예: "15일 남음")
- 환자명 (이환자 (68세, 남))
- 태그들: 폐암 3기, 식사 가능, 배변 도움 필요

### 2. 기본 정보 섹션
| 필드 | 예시 |
|------|------|
| 생년월일 | 1945.12.12 |
| 키 | 173cm |
| 몸무게 | 60kg |

### 3. 환자 상태 섹션
| 필드 | 예시 |
|------|------|
| 진단명 | 폐렴 |
| 식사 도움 | 부분적 도움 필요 |
| 거동 상태 | 부족 필요, 지팡이 사용 |
| 요청 사항 | 계단 이용 불가 |

### 4. 보호자 섹션
| 필드 | 예시 |
|------|------|
| 이름 | 나보호 |
| 휴대폰 | 010-1234-5678 |
| 관계 | 자녀 |

### 5. 간병 위치 섹션
- 병원명 및 주소
- 상세 주소 (동/호수)

### 6. 간병 기간 섹션
- 전체 기간 및 일수
- 근무 요일
- 근무 시간

### 7. 하단 CTA 버튼
- "간병 일지 작성하기" (고정)
